TESTING ACCURACY

Creating a best prediction column that find the highest probability classification for every event based on the model’s predictions.

comparison <- cbind(cleaned_data$total_bases, cleaned_data_predictions_prob)
comparison_best_results <- comparison |>
  mutate(best_prediction = case_when(
    .pred_0 == pmax(.pred_0, .pred_1, .pred_2, .pred_3, .pred_4) ~ 0,
    .pred_1 == pmax(.pred_0, .pred_1, .pred_2, .pred_3, .pred_4) ~ 1,
    .pred_2 == pmax(.pred_0, .pred_1, .pred_2, .pred_3, .pred_4) ~ 2,
    .pred_3 == pmax(.pred_0, .pred_1, .pred_2, .pred_3, .pred_4) ~ 3,
    .pred_4 == pmax(.pred_0, .pred_1, .pred_2, .pred_3, .pred_4) ~ 4
  ))

Creating a match column where a 1 indicates that the best_prediction of the model and the real value are the same and 0 is where they do not match.

accuracy <- comparison_best_results |>
  mutate(match = case_when(
    `cleaned_data$total_bases` == best_prediction ~ 1,
    TRUE ~ 0
  ))

Calculate the percentage of total bases correctly predicted.

amount_correctly_predicted <- accuracy |>
  count(match == 1)
amount_correctly_predicted_percentage <- (amount_correctly_predicted$n[2] / (amount_correctly_predicted$n[1] + amount_correctly_predicted$n[2])) * 100

CALCULATING xwOBACON

Weights for 0,1,2,3,4 total bases for 2024 season

woba_2024_weights <- c(0, 0.882, 1.254, 1.590, 2.050)
total_bases_probs <- cleaned_data_predictions_prob[c('.pred_0','.pred_1','.pred_2','.pred_3','.pred_4')]
calculate_xwOBACON <- function(x) {
  return(sum(x * woba_2024_weights))
}
xwOBACON <- total_bases_probs |>
  mutate(xwOBACON = apply(X = total_bases_probs, MARGIN = 1, FUN = calculate_xwOBACON))

CALCULATING xwOBA

Importing dataset that includes all events, not just ones that were batted into play.

Getting distinct values to be able to left-join the dataset with all events with our calculated xwOBACON values.

xwOBACON_w_LA_and_LS <- cbind(xwOBACON, cleaned_data$launch_angle, cleaned_data$launch_speed)

xwOBACON <- xwOBACON_w_LA_and_LS |>
  select(`cleaned_data$launch_speed`, `cleaned_data$launch_angle`, xwOBACON) |>
  distinct()

Joining all events dataset with the dataframe where we have calculated xwOBACON. This ensures that the xwOBA values for walks, hit by pitch, strikeout, and strikeout_double_play are correct. This is important because these values play a role in the final calculation of xwOBA.

final_df <- final_df |>
  left_join(xwOBACON, by = c("launch_speed" = "cleaned_data$launch_speed",
                             "launch_angle" = "cleaned_data$launch_angle")) |>
  mutate(xwOBA = case_when(
    events == 'walk' ~ 0.689,
    events == 'hit_by_pitch' ~ 0.720,
    events == 'strikeout' ~ 0,
    events == 'strikeout_double_play' ~ 0,
    TRUE ~ xwOBACON))

Calculating xwOBA for players with more that 100 plate appearances.

cleaned_data_grouped_xwOBA <- final_df |>
  group_by(batter) |>
  #filter(n() > 100) |>
  summarize(xwOBA = mean(xwOBA), woba = mean(woba))

Plotting predicted xwOBA vs official MLB values. Ideally, if we graph the official values on the x-axis and out predicted values on the y-axis, then there should be a perfect 1-to-1 linear fit. The adjusted R^2 value of 0.92329 indicates that there is a significant linear fit between the official and our predicted values. Considering an R^2 of 1.0 is a perfect replica of the official values, I would say that our model is a decent open-source version of the xwOBA Statcast model.
