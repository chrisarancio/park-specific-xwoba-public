  This project intends to create an open-source version of the Statcast xwoba model. We will train Random Forest, Extreme Gradient Boosting, and Generalized Additive Model (GAM) algorithms using official Statcast data. The best performing model will then be used in an analysis of a player's park-specific xwoba, as opposed to their officially published values for the overall season.
  This will be done by taking the xwOBA predictions for all events and subsetting them by player and ballpark before taking the average. This research would aim to answer questions like: 1. What players have the highest modeled xwOBAs at each ballpark? 2. How do the modeled xwOBAs compare to the official values? 3. Is there any correlation between these modeled xwOBAs and the Ballpark Factor metric?
